# Prompts for DALL-E 3 image generation: Fallout Clicker Shop Icons

Each prompt is for a distinct item icon fitting the game's retro-futuristic, post-apocalyptic, green phosphor CRT aesthetic.

---

### "No Ads" Icon
**Prompt:** `A pixel art or retro-futuristic icon of a classic television set with a glowing green CRT screen being smashed by a heavy wrench. The style should be monochromatic green on black, resembling an old MS-DOS game or Fallout Pip-Boy interface, highly contrasted, no background.`

### "Starter Pack" Icon
**Prompt:** `A pixel art or retro-futuristic icon of a worn canvas backpack resting next to a steel pickaxe, a rusty water canteen, and a sealed ration tin. The style should be monochromatic green on black, resembling an old MS-DOS game or Fallout Pip-Boy interface, highly contrasted, no background.`

### "Premium Caps (Wallet)" Icon
